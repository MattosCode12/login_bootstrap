Enzo Nicoletti de Mattos

Desenvolvimento de uma tela de login estática utilizando HTML e Bootstrap, com foco em estruturação de formulário, responsividade e estilização por meio de classes prontas do framework. A interface contém campos de e-mail e senha, botão de login e links auxiliares, apresentando layout organizado, acessível e adaptável para diferentes tamanhos de tela.
